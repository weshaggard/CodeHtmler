CodeHtmler plugin for Windows Live Writer
Author Wes Haggard http://puzzleware.net/codehtmler

The plugin allows for highlighted code to be inserted into blog posts.


Requirements:
.Net Framework 2.0
Windows Live Writer


Install Plugin:

1) Shutdown Windows Live Writer

2) Run install.bat which will copy the following files into %Program Files%\Windows Live\Writer\Plugins\
CodeHtmler.dll 
CodeHtmler.LiveWriterPlugin.dll 

3) Start Windows Live Writer and you should see an "Insert Code" option.
